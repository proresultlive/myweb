📖 എങ്ങനെ ഉപയോഗിക്കാം
🧑‍🤝‍🧑
1. PARTICIPANTS
ആദ്യം Participants section-ലേക്ക് പോകുക. അവിടെ ഓരോ participant-ന്റെ പേര്, ചെസ്റ്റ് നമ്പർ, കാറ്റഗറി, ഗ്രൂപ്പ് എന്നിവ ചേർക്കുക.
🎯
2. PROGRAMS
Programs section-ൽ ആദ്യം "+ Category" ക്ലിക്ക് ചെയ്ത് Arts, Sports തുടങ്ങിയ categories ഉണ്ടാക്കുക. പിന്നെ "+ Program" ക്ലിക്ക് ചെയ്ത് programs ചേർക്കുക.
👥
3. TEAMS
Teams section-ൽ ഓരോ team-ന്റെ പേര് edit ചെയ്ത് സ്വന്തം team name കൊടുക്കാം. Members-നെ Programs → Scores വഴി add ചെയ്യാം.
📝
4. SCORES
Programs page-ൽ ഓരോ program-ന്റെ "📝 Scores" button ക്ലിക്ക് ചെയ്ത് participant-ന്റെ പേരും score-ഉം enter ചെയ്യുക.
🏆
5. RESULTS
മുകളിലെ "🏆 Calculate" button ക്ലിക്ക് ചെയ്ത് results കാണാം. Podium, leaderboard, category breakdown എല്ലാം automatically കാണിക്കും.
⚙️
6. SETTINGS
Admin മാത്രം Settings access ചെയ്യാം. അവിടെ password change ചെയ്യാനും additional users manage ചെയ്യാനും സാധിക്കും.
